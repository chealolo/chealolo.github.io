Github hexo 블로그 만들기 !

IT회사에 처음 입사한후 많은 변화가 생겼다.
일단은 블로그를 시작한점과 Javascript , 중국어 공부를 시작했다는 점이다.
사수의 권유로 블로그를 만들게 되었는데 GitHub는 처음이라 굉장히 많은 어려움을 겪었다.

ERROR ERROR ERROR!!

처음에는 지킬테마로 시작해보았는데 스타일이 들어오지않아 실패하고
두번째에는 _config.yml 파일과 package.json 파일을 삭제해버리고..
세번째는 index.html 파일이 없어지는 사태가 발생해버렸다 ..!
이러한 시행착오를 겪고나서 극복한결과 테마 적용과 글쓰는것에 성공했다 !

왜 Hexo 인가?

블로그 프레임워크 비교

  1. Hexo
  2. Hugo
  3. Jekyll

  • Hexo

    • Javascript기반으로 만들어졌다.
    • 마크다운 지원

  • Hugo

    • Go 기반으로 만들어 졌다.

  • Jekyll

    • Ruby기반으로 만들어졌다.

사실상 이제막 시작해서 뭐가뭔지 어리둥절하다. Jekyll 로 처음에 만들다가 실패해서 사수가 Hexo를 추천해주셨다. 세가지의 프레임워크는 다 각각의 장단점이 있는듯하다.

Hexo 설치하기 !

저같은 경우는 회사에서 npm을 사용하지않고 yarn을 사용해서 yarn으로 설치를 했습니다.

$ yarn install -g hexo -cli
$ hexo init 본인폴더명 지정
$ cd 본인이 지정한 폴더명
$ yarn install




이렇게 만들게되면 내가만든 기본 블로그를 로컬서버로 확인해보실수 있습니다 !
